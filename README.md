# 📊 日商簿記3級 模擬試験 Web App

这是一个基于 Web 的日商簿记3级模拟考试应用，专为练习仕訳（会计分录）设计。它模拟了真实的考试逻辑，支持随机抽题与即时评分。

🔗 **在线演示 (Live Demo)**: [https://Joe-Xuu.github.io/boki/](https://Joe-Xuu.github.io/boki/)

## ✨ 主要功能

* **🔀 随机抽题模式**: 每次考试从总题库中随机抽取 **15道** 题目，打破固定顺序记忆，模拟真实考场压力。
* **🎯 动态勘定科目**: 智能下拉菜单逻辑，根据当前题目仅显示相关的备选科目（语群），还原真实试卷体验。
* **📝 实战输入界面**: 模拟借方/贷方分录输入，支持动态增加行数以处理复合仕訳（多对多或一对多交易）。
* **⚡️ 即时反馈**: 提交答案后立即显示正误判定，并提供详细的题目解析和正确分录对比。
* **🔁 随时重开**: 支持中途打断考试并重新抽取新题。

## 🛠️ 技术栈

该项目采用纯原生技术构建，轻量且无需构建工具：

* **HTML5 & CSS3**: 语义化标签与布局。
* **Vanilla JavaScript (ES6+)**: 核心逻辑（洗牌算法、状态管理、DOM 操作）。
* **Tailwind CSS (CDN)**: 快速构建响应式、现代化的用户界面。
* **Phosphor Icons**: 简洁直观的图标库。

## 🚀 如何运行
本地运行
克隆或下载本项目到本地。

直接使用浏览器打开 index.html 文件即可运行，无需安装 Node.js 或配置服务器。

部署到 GitHub Pages
将项目文件夹上传至 GitHub 仓库（例如作为 boki 子目录）。

确保仓库已开启 GitHub Pages 功能。

通过 https://<username>.github.io/boki/ 访问。
